2. 网络测试

2.1 测试过程描述

2.1.1 虚拟机申请过程

在华为云上申请 4 台虚拟机，分别位于

  • 华北-乌兰察布一 可用区2
  • 华北-乌兰察布一 可用区2
  • 华北-乌兰察布一 可用区3
  • 西南-贵阳一 可用区1

系统均为CentOS 8.2，1vCPUs，内存 4 GiB

带宽均为 5 Mbit/s，按流量计费

2.1.2 测试工具

使用iperf3

2.1.3 操作过程

因不同区的安全组不通用，所以在华北-乌兰察布一、西南-贵阳一中的被使用的安全组入方向规则中添加5000端口，即允许任何 ip 访问以上4台虚拟机的5000端口

本人采用MobaXterm软件远程连接服务器，每个虚拟机通过yum install iperf3安装iperf3

可以通过iperf3 -v查看是否安装成功

通过命令iperf3 -s -i 5 -p 5000可以作为iperf3的服务端

通过命令iperf3 -c ServerIP -i 5 -t 100 -p 5000可以作为iperf3的客户端

2.2 跨区域公网

虚拟机网络配置情况

两台虚拟机分别位于西南-贵阳一 可用区1、华北-乌兰察布一 可用区3

ip地址分别为140.210.207.189，116.204.121.131

两台虚拟机带宽均为5Mbit/s

西南-贵阳一 作为服务端，华北-乌兰察布一 作为客户端

测试结果

客户端sender Bitrate为 5.54 Mbits/sec

客户端receiver Bitrate为 5.30 Mbits/sec

服务端receiver Bitrate为 5.30 Mbits/sec

经测试，跨区域无法使用私网Ping通

2.3 同区域跨可用区公网+私网

2.3.1 公网

虚拟机网络配置情况

两台虚拟机分别位于华北-乌兰察布一 可用区2、华北-乌兰察布一 可用区3

ip地址分别为116.204.123.135，116.204.121.131

两台虚拟机带宽均为5Mbit/s

乌兰察布一 可用区2 作为服务端，乌兰察布一 可用区3 作为客户端

测试结果

客户端sender Bitrate为 5.59 Mbits/sec

客户端receiver Bitrate为 5.50 Mbits/sec

服务端receiver Bitrate为 5.50 Mbits/sec

2.3.2 私网

虚拟机网络配置情况

两台虚拟机分别位于华北-乌兰察布一 可用区2、华北-乌兰察布一 可用区3

私有ip地址分别为192.168.0.151，192.168.0.139

两台虚拟机带宽均为5Mbit/s

乌兰察布一 可用区2 作为服务端，乌兰察布一 可用区3 作为客户端

测试结果

客户端sender Bitrate为 554 Mbits/sec

客户端receiver Bitrate为 553 Mbits/sec

服务端receiver Bitrate为 553 Mbits/sec

私网速度远比公网快
